# Cloud DevOps CI/CD Project

## Project Overview
This project demonstrates a basic Cloud Computing and DevOps CI/CD workflow.
A simple web application is version-controlled using GitHub and will be
automated using CI/CD tools in later stages.

## Technologies Used
- Cloud Computing (AWS)
- DevOps
- Git & GitHub
- HTML

## Project Structure
- index.html : Simple web application file

## Future Enhancements
- CI/CD pipeline using Jenkins
- Docker containerization
- Cloud deployment on AWS EC2
- Monitoring and logging
